# ZB-HOST Ultimate Multi-User Bot Management Panel

## Features
- Multi-user system with unique User IDs
- Admin panel with user management (Ban/Unban)
- Bot deployment (.py and .zip support)
- Bot scheduler
- API keys generator
- Notes & Favorites
- File manager
- Bot templates
- Real-time CPU/RAM stats
- Mobile responsive

## Default Login
- Username: `admin`
- Password: `admin123`

## Deploy on Netlify
1. Drag and drop this folder to netlify.com/drop
2. Done!

## Tech Stack
- HTML5
- Tailwind CSS
- Chart.js
- Font Awesome
- LocalStorage (No backend required)
